  ## ABOUT TOOL :

UPI-INT is an indian osint tool based on recon on upi for information gathering and analysis.


## AVAILABLE ON :

* Termux
* Windows
* Linux
  
 ### Prerequisites

- Git and node

### Installation

- Clone the Repo
  ```
  git clone https://github.com/BiswajeetRay7/UPI-INT
  ```
- Open cloned directory
  ```
  cd UPI-INT
  ```
- Install Dependencies 
  ```
  npm install 
  ```
- Now Run Tool
  ```
  npm start
  ```
